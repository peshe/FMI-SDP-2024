# СДП Семинар 12 -- Пирамида. Приоритетна опашка. Още за графи

## Пирамиди. Двоична пирамида

| Операция                    | Сложност |
| --------------------------- | -------- |
| Добавяне на елемент         | O(log n) |
| Изтриване (Extract Min/Max) | O(log n) |
| Достъп до min/max           | O(1)     |
| Построяване                 | O(n)     |
